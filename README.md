# BombShell

A simple shell built using Rust.

### Getting started

* clone the repo:
  ```bash
  $ git clone https://github.com/ro6ley/BombShell.git
  $ cd BombShell
  ```

* run the shell:
  ```bash
  $ cargo run
  >> 
  ```

* enter bash commands as you would on any other shell, e.g
  ```bash
  >> ls
  Cargo.lock      Cargo.toml      README.md       src             target
  ```
